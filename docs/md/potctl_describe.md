## potctl describe

Get detailed information of an existing resources

### Synopsis

Get detailed information of an existing resources.
 
Most resources require a working Controller in the Namespace in order to be described.

### Options

```
  -h, --help                 help for describe
  -o, --output-file string   YAML output file
```
